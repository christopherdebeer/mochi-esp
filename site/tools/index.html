Author device sprites end to end: create substrate sheets, preview cells, draw tap zones with typed actions (event / nav / talk-seed), tune the per-type line-screen, and export MPK1 packs — all on the same encode.ts / mpk.ts the firmware reads. Cells stream from mochi.val.run/devsprite; sheets + zones persist in substrate.
