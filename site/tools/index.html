SPRITE·FORGE

→

Slice a sprite sheet into a grid, resample each cell, and pack to a single .bin (1bpp / gray8 / rgb565) with a drop-in C reader and authored per-sprite tap rectangles. Halftone / dither effects (Floyd–Steinberg, Atkinson, Bayer, line-screen) for 1-bit panels.

note — SPRITE·FORGE emits a generic SPRT pack (single-plane, bit 1 = ink). mochi-esp's own device wire format differs: bit 0 = ink, and cells carry a second mask plane for transparency. See design/05-sprite-format.md. Use this tool for general ESP32 / Arduino projects, or as a starting point for device assets you convert to the two-plane format.
